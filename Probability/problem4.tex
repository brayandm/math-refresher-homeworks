\textbf{Problem 4:}

\singlespacing

Let $P$ be a distribution on $\mathbb{R}$ and $F$ be its distribution function. Are
the following equalities true? $F(a-)$ denotes one-sided limit where function argument
approaches a from the left.

\begin{enumerate}
    \item $P([a, b]) = F(b) - F(a -)$
    \item $P((a, b)) = F(b) - F(a)$
    \item $P([a, b)) = F(b -) - F(a -)$
    \item $P(\{x\}) = F(x) - F(x -)$
\end{enumerate}
